# Blog image credits & licensing

Standing rule: every blog post and informative page gets a relevant, **freely-licensed** image with proper credit, researched at build time. Prefer **public domain**; otherwise CC with attribution.

**Important — current images are HOTLINKED from Wikimedia Commons** (via `Special:FilePath`), because files can't be downloaded into the project automatically. **Before public launch, download each file and self-host it under `/assets/blog/`** (then swap the `src`), keeping the credit captions below. Hotlinking is fine for now but self-hosting is faster and more reliable.
